# 工资查询系统使用说明

## 一、系统功能

1. **员工查询**：输入姓名 + 银行卡号后6位，查询自己的工资记录
2. **管理员上传**：登录后上传Excel工资表，系统自动识别工资项目

## 二、安装步骤

### 1. 安装Python
- 访问 https://www.python.org/downloads/
- 下载并安装Python 3.8或更高版本
- **重要**：安装时勾选"Add Python to PATH"

### 2. 安装依赖包
```bash
pip install -r requirements.txt
```

### 3. 启动系统
```bash
python app.py
```
或双击 `启动.bat`

### 4. 访问系统
- 员工查询：http://localhost:5000
- 管理员入口：http://localhost:5000/admin
- 默认用户名：`admin`
- 默认密码：`admin123`

## 三、Excel格式要求

上传的Excel文件需要包含以下列：

| 姓名 | 银行卡号 | 基本工资 | 岗位津贴 | ... |
|------|----------|----------|----------|------|
| 张三 | 123456   | 5000     | 2000     | ... |

- **必需列**：`姓名`、`银行卡号`
- **可选列**：其他列会自动识别为工资项目
- **文件格式**：.xlsx 或 .xls

## 四、使用流程

### 管理员操作流程：
1. 登录管理员页面
2. 选择月份（如：2024-01）
3. 选择Excel文件
4. 点击上传

### 员工查询流程：
1. 打开查询页面
2. 输入姓名（如：张三）
3. 输入银行卡号后6位（如：123456）
4. 点击查询

## 五、常见问题

### Q: 忘记管理员密码怎么办？
A: 重新运行程序，会自动重置为 admin/admin123

### Q: 上传失败怎么办？
A: 检查Excel是否包含"姓名"和"银行卡号"列

### Q: 如何查看历史记录？
A: 员工查询会显示所有月份的历史工资记录

## 六、部署到公网（可选）

### 方案1：使用云服务器
1. 购买云服务器（如阿里云、腾讯云）
2. 安装Python和依赖
3. 上传项目文件
4. 运行：`python app.py`
5. 配置域名解析

### 方案2：使用PAAS平台
- Heroku、Render、 Railway 等
- 直接上传代码即可运行

### 配置域名：
1. 购买域名
2. 在域名服务商处添加A记录，指向服务器IP
3. 在服务器上配置nginx反向代理
